# Instructions pour dépanner « disque protégé en lecture seule »

## Ce guide concerne

Microsoft Windows 7+

## Étapes

1. Accéder <a href="https://raw.githubusercontent.com/Ron-RONZZ-org/ronzz-CDN-2/refs/heads/main/skriptoj/komputilaj-iloj/MS-Win/disko-nurlegebla-protekto-fr.ps1" target="_blank">le script `disko-nurlegebla-protekto-fr.ps1`</a> sur le répertoire Github.  
1. Copie tout le contenu avec `Ctrl+A`, puis `Ctrl+C`.  
1. Appuie sur la touche `WIN` pour ouvrir le menu de démarrage de Windows.  
1. Recherche « Powershell » dans le menu de démarrage.  
1. Choisis l’option « Ouvrir Powershell en tant qu’administrateur ».  
1. Confirme l’ouverture en cliquant sur « Oui » dans la fenêtre d’avertissement.  
1. Une fois Powershell ouvert, colle le script avec `Ctrl+V`.  
1. Appuie sur `Entrer`.  
1. Suis les instructions affichées à l’écran pour terminer le dépannage.
